Usabilidad en la web    

¿Que es Usabilidad?  

La usabilidad se refiere a la facilidad con la que un usuario  de la web puede acceder  ya sea en la navegacion o interaccion en un sitio web.         

Un sitio web  debe cumplir con una serie de criterios, por ejemplo:

Facilidad y rapidez 

Efectividad  

Memorabilidad  

Satisfaccion  

Storytelling

Inicio  

Sofia como usuario de  internet , esta en busca de un producto y no sabe en que sitio web comprar, por lo tanto entra a un sitio web y este no funciona de manera correcta, ya que los botones nos responden y los textos son dificiles de leer. 

Si la pagina esta mal diseñada sera muy probable que se abandone la pagina web.

 

 

Desarrollo 

Sofia decide entrar a otro sitio web, todo se encuentra mas claro, el menu es intuitivo y la informacion es correcta. 

Si el sitio web cumple con los criterios  los usuarios podran cumplir sus objetivos dando un solo clic y sin complicaciones. Esto genera ua experiencia positiva  y esto  motiva a  regresar al sitio web.

 

Desenlace  

Gracias a la web diseñada de manera correcta  Sofia puede navegar de manera segura y facil. 

 La usabilidad no solo mejora la  experinecia del usuario, tambien favorece las conversaciones, interacciones    y fidelidad del usuario . Tambien beneficia a la empresa ya que puede revisar el aumento de  trafico  y ventas.
